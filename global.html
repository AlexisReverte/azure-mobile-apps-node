apiDefinition

Specifies an api

Properties:
Name Type Default Description
authorize boolean false

Execute the authorize middleware for each api operation.
get function | Array.<function()>

Middleware to execute on get requests to this api. You can also set the authorize/disable properties on this member.
post function | Array.<function()>

Same as get, but for post requests.
put function | Array.<function()>

Same as get, but for put requests.
patch function | Array.<function()>

Same as get, but for patch requests.
delete function | Array.<function()>

Same as get, but for delete requests.
Source:
  • jsdoc/api.js, line 4
Example
var api = module.exports = {
     // get middleware
     get: function (req, res, next) {
         res.status(200).end();
     },
     
     // patch middleware
     patch: return204,

     // array of post middleware
     post: [addHeader, return204]

     // Require authorisation for all operations
     // authorize = true;
};

// Require authorisation for patch
api.patch.authorize = true;

// Disable the post method
api.post.disable = true;

function return204(req, res, next) {
     res.status(204).end();
}

function addHeader(req, res, next) {
     res.set('post', 'true');
     next();
}
