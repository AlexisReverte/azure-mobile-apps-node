azure-mobile-apps

The azure-mobile-apps module is the Nodejs implementation of Azure Mobile Apps

(require("azure-mobile-apps"))(configuration) → {express}

Creates an instance of the azure-mobile-apps server object for the platform specified in the configuration. Express 4.x is currently the only supported platform.
