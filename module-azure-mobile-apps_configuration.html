Module: azure-mobile-apps/configuration

Azure Mobile Apps can be configured by providing values for specifiec environment variables. These variables are described below. Variable names are not case sensitive. For configuration using Javascript, see global configuration.
Parameters:
Name Type Description
MS_MobileAppName string Name of the mobile app
MS_MobileLogLevel string Minimum log level of messages to log (debug, info, warn, error)
MS_DebugMode boolean Enables or disables debug mode
MS_TableConnectionString string Connection string to use to connect to SQL Server
MS_DynamicSchema boolean Disables dynamic schema for tables when set to false
EMA_RuntimeUrl string Authentication gateway URL
MS_SigningKey string JWT token signing / validation key
MS_CrossDomainWhitelist string Comma delimited list of domains to allow cross domain requests from
MS_NotificationHubName string Name of the notification hub for the app
MS_NotificationHubConnectionString string Connection string to notification hub for the app
