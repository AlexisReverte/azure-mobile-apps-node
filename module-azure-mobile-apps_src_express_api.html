azure-mobile-apps/src/express/api

This module contains functionality for adding custom apis to an Azure Mobile App. It returns a router that can be attached to an express app with some additional functions for registering apis.

(require("azure-mobile-apps/src/express/api"))(configuration)

Create an instance of an express router for routing and handling api requests.
